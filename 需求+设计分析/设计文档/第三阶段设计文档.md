# 顺序图描述文档

顺序图描述了系统类，用户类的写作关系</br>
顺序图如下：

```sequence
title: 顺序图

participant 用户
participant 出题系统


用户->出题系统: 输入题目数目
用户->出题系统: 用户选择乘方模式
出题系统->出题系统: 生成初始算式
出题系统->出题系统: 进行判重以及除以0的操作 发现重复，重新生成
出题系统->出题系统: 生成成功
出题系统->用户: 显示所出题目
用户->出题系统: 提交答案
出题系统->出题系统: 判断答案正确数目
出题系统->用户: 将正确数目显示给用户
出题系统->用户: 是否显示历史记录
用户->出题系统: yes
出题系统->用户: 显示历史记录
用户->出题系统: no
出题系统->用户: 结束答题

```
